## [0.0.0] - 2026-04-30

### Added
- Initial project structure.
- Support for Kotlin DSL (`build.gradle.kts`).
- Support for Groovy DSL (`build.gradle`).
- Support for Version Catalogs (`libs.versions.toml`).
- Support for `gradle.properties`.
- Cooldown window for supply-chain security.
- Interactive mode with TUI picker.
- JSON output mode.
